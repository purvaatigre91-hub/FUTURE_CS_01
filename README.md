## Security Alert Monitoring & Incident Response

### Overview
This project demonstrates monitoring security alerts using Splunk SIEM, identifying suspicious activity, classifying incidents, and preparing an incident response report.

### Tools Used
- Splunk SIEM
- Sample Syslog Files

### Key Findings
- Ransomware and Trojan malware detected
- Multiple failed login attempts identified
- Suspicious external file access observed

### Skills Demonstrated
- Log Analysis
- Alert Triage
- Incident Classification
- SOC Operations
